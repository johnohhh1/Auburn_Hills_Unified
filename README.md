### **Prerequisites**
```bash

npm or yarn

```

### **Installation**
```bash
# Clone the repository
git clone https://github.com/johnohhh1/Auburn_Hills_Unified.git

# Navigate to the project
cd Auburn_Hills_Unified

# Install dependencies
```
